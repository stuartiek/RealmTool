RealmTool Plugin Documentation

This document describes all features, commands, and user interfaces exposed by the RealmTool Minecraft plugin.

1. Getting Started

Installation:

  1. Drop the compiled JAR into your server's plugins/ folder.
  2. Start or restart the server to generate the default data.yml / config structure.
  3. Adjust your permissions and configuration (if desired) using the generated data.yml file.

Note: The plugin stores persistent data in data.yml (rank definitions, NPC library, warp/home locations, etc.).

2. Command Overview

The primary command is /dmt. Many other commands are provided by the plugin as well.

2.1 Core Commands

Most /dmt subcommands can be secured using permission nodes in the form dmt.command.<subcommand> (e.g. dmt.command.menu).

Common permission nodes:

  • dmt.command.menu - open the main menu (player + admin panel)
  • dmt.command.sethub - set the hub teleport location
  • dmt.command.antlag - control automatic cleanup
  • dmt.command.rank - manage ranks
  • dmt.command.npc - manage NPC library and spawning
  • dmt.command.summon - summon an NPC from the library
  • dmt.command.list - list NPCs (/dmt list npcs)
  • dmt.command.documentation - generate and download the documentation PDF
  • dmt.command.tpjail, dmt.command.punish, etc. are also supported for specific subcommands
/dmt help
/dmt menu
/dmt sethub
/dmt unsethub
/dmt antlag <on|off|now>
/dmt rank <create|remove|list|info|add|addprefix|addperm> ...
/dmt npc <list|add|remove|summon>
/dmt summon <library-name>
/dmt list npcs
/dmt documentation

2.2 Player Convenience Commands

/kit         - Open the kit browsing & purchasing interface
/hub         - Teleport to the server hub (saved via /dmt sethub)
/balance     - View your coin balance and XP level
/balance <player>           - (admin) view another player's balance
/balance <player> <set|add|remove> <amount> - (admin) modify balance

2.3 Admin Commands (require permission dmt.admin)

/dmt menu           - Open the main menu (player + admin panel)
/dmt setpunishloc   - Set the punishment teleport location
/dmt setjailloc     - Set the jail teleport location
/dmt unsethub        - Remove the stored hub teleport location
/dmt tpjail         - Teleport yourself to the jail location
/dmt punish <player> <duration> - Punish a player (teleport + temporary restrictions)
/dmt tp world <world> - Teleport to another world (respects world lock)

2.4 NPC Library

RealmTool includes a small NPC skin library system used by /dmt summon and the menu NPC spawner.

/dmt npc list                     - Show stored NPC skin names
/dmt npc add <username>        - Add a player skin to the NPC library (uses skinstealer)
/dmt npc remove <username>     - Remove a stored NPC skin
/dmt npc summon <library-name> - Summon an NPC using a stored skin
/dmt summon <library-name>     - Shortcut to summon a stored NPC

2.5 Anti-lag Cleanup

The plugin can periodically clear item drops to reduce lag.

/dmt antlag on      - Enable automatic cleanup
/dmt antlag off     - Disable automatic cleanup
/dmt antlag now     - Run cleanup immediately

3. User Interfaces

3.1 Main Menu

Use /dmt menu to open the main menu. From here you can open either the Player Menu or the Admin Menu.

3.2 Player Menu

The player menu provides quick access to common utilities.

  • Set Home – Save your current location as a personal home.
  • TP Home – Teleport to your saved home.
  • Set Warp – Save a named warp location.
  • View Warps – Browse and teleport to saved warps.
  • Tickets – Open the ticket system to create support requests.
  • Chunk Claims – Open the claim management UI.
  • Players (TPA) – View online players and send TPA requests.
  • Kits – Browse & purchase kits (XP-cost based).
  • Report Player – Create a report for staff.
  • Custom Enchantments – View unlocked enchants (unlocked via quests).

3.3 Admin Menu

The admin menu is gated by permissions and provides: weather control, world locking, player utilities, broadcasting, and more.

Admin Menu Buttons & Permissions

  • adminpanel.weather.clear – Clear current weather.
  • adminpanel.weather.rain – Force rain.
  • adminpanel.weather.thunder – Force thunder.
  • adminpanel.netherlock – Toggle Nether world lock.
  • adminpanel.endlock – Toggle End world lock.
  • adminpanel.worldsettings – Open world settings submenu.
  • adminpanel.playerdirectory – View & manage players (TPA list / actions).
  • adminpanel.gamemode.creative – Set yourself to Creative.
  • adminpanel.gamemode.survival – Set yourself to Survival.
  • adminpanel.gamemode.spectator – Set yourself to Spectator.
  • adminpanel.heal – Heal yourself (restore health + hunger).
  • adminpanel.punished – View punished players list.
  • adminpanel.inspector – Enable player inspector tools.
  • adminpanel.broadcast – Send a broadcast message to all players.
  • adminpanel.tickets – View and manage open tickets.
  • adminpanel.worldutilities – Access world utilities (e.g., fast teleport, world actions).
  • adminpanel.events – Trigger or manage server events.
  • adminpanel.hologram – Open the hologram wand UI.

World Settings Submenu

From the world settings submenu, admins can:

  • Lock/unlock specific worlds (Nether/End).
  • Manage global world behavior (weather, time, etc.)

4. Rank System

The plugin supports a flexible rank system with prefixes and permission sets. Ranks are stored in data.yml under ranks.

Commands

/dmt rank list
/dmt rank info <rank>
/dmt rank create <rank>
/dmt rank remove <rank>
/dmt rank add <rank> <player>
/dmt rank <rank> add <player>
/dmt rank addprefix <rank> <prefix>
/dmt rank <rank> addprefix <prefix>
/dmt rank addperm <rank> <permission>
/dmt rank <rank> addperm <permission>

Example:

/dmt rank Admin addperm adminpanel.broadcast

Rank Behavior

  • Ranks can have a prefix which is shown in chat, player nametag, and tab list for members.
  • Ranks can include a list of permissions; those permissions are applied via a temporary attachment.
  • Players may be assigned a rank by UUID or by membership list in the rank definition.

5. NPC / Hologram Utilities

The plugin includes NPC summoning and hologram tools.

  • NPC Library – Use /dmt npc add <username> to store a player skin, then spawn it via /dmt npc summon <username>.
  • Hologram Wand – Available from the admin menu (permission adminpanel.hologram).

6. Anti-Xray & Anti-Lag

The plugin includes anti-Xray analysis and item cleanup features to help stabilize servers:

  • Anti-Xray runs in the background and warns players breaking ores too fast.
  • Anti-lag cleanup periodically removes dropped items (configurable via /dmt antlag).

7. Configuration / Data File

All plugin state is stored in data.yml. Key sections include:

  • ranks – rank definitions (prefix, permissions, members)
  • npcLibrary – stored NPC skins
  • homes, warps, hub_location – teleport locations
  • appeals – ticket system storage
  • anti_lag – anti-lag enabled flag and timers

Tip: When editing data.yml manually, keep YAML structure intact and restart the server after changes.
